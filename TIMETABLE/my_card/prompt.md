# 초기 프롬프트

시간표 작성앱을 만들어줘.

[무엇을] 코딩 경험이 없는 초보자용으로, ‘내 소개 카드’ 웹페이지를 만들어줘.
[조건] 파일은 index.html 과 style.css, script.js로 나눠줘. 외부 라이브러리는 쓰지 말아줘.
[확인] 브라우저에서 index.html을 열면 바로 보이게 해줘.
[범위] my_card 폴더 안의 파일을 수정해서 만들어줘.
[마무리] 실행 방법(어떤 파일을 열면 되는지)과 수정 포인트 3가지를 알려줘.
[스타일] 흰색 바탕에 심플한 느낌으로. 그리고 아기자기한 파스텔톤으로 꾸미기

[시간표 내용]
- 1교시는 9시이고 1시간 단위로 9교시까지 만들어줘. 
- 교시와 시간을 같이 표시해줘 (ex. 1교시(9~10))
- 1시간 단위로 입력, 시간표에 표시도 되게 해줘. 
-시간표 옆에 강의명, 강의실, 수업 시간, 학점 등을 입력하는 폼도 만들어줘.

# 1차 수정
그 23 학점이상인 수업은 한 시간 단위가 아니니까 한 번에 2시간 이상 한번에 표에 넣을 수 있게 수정해줘. 또한 지금 하루 시간표인데 월금 까지의 시간표로 만들어줘

# 2차 수정
제목을 '김연수의 시간표'를 영어로 고쳐줘. index.html 만 수정해줘.

# 3차 수정
글씨가 안 보여. 그리고 조금 더 빈티지하게 수정해줘. 약간 빈티지와 레트로 사이

# 4차 수정
생각해보니까 3학점 3시간 수업이여도. 월요일 2시간 화요일 1시간 해서 3학점인 경우가 있던데 그런 점도 참고해줘

# 5차 수정
그 추가된 시간표에 추가된 강의를 보니까 너무 다른 강의와 가시성있게 구별하기 어려워. 블록처럼 색을 다르게 해주던가 조금 수정해줘
