# 📚 Kim Yeon-su's Timetable

대학 시간표를 효율적으로 관리할 수 있는 웹 기반 시간표 애플리케이션입니다.

## ✨ 주요 기능

- **주간 시간표 관리** - 월요일부터 금요일까지의 일정을 한눈에 확인
- **학점 기반 강의 배치** - 학점에 따라 자동으로 시간 할당 (예: 3학점 = 3시간)
- **다중 날짜 강의 분산** - 한 강의를 여러 날에 나누어 배치 (예: 월 2시간 + 화 1시간 = 3학점)
- **시각적 색상 구분** - 강의별로 다른 색상으로 표시하여 구분 용이
- **실시간 학점 계산** - 현재 등록한 총 학점 자동 계산
- **데이터 자동 저장** - LocalStorage를 이용한 자동 저장 및 불러오기

## 🎨 특징

- **빈티지 스타일** - 따뜻한 갈색/베이지 색상의 레트로한 디자인
- **직관적 UI** - 복잡한 기능 없이 간단하고 쉬운 사용자 인터페이스
- **반응형 디자인** - 다양한 화면 크기에 대응
- **외부 라이브러리 불필요** - HTML, CSS, 순수 JavaScript만 사용

## 🚀 사용 방법

1. `index.html`을 웹 브라우저에서 열기
2. 강의명, 강의실, 학점 입력
3. 각 날짜별로 시작 교시와 수강 시간 설정
4. "강의 추가" 버튼으로 시간표에 추가
5. 강의를 클릭하면 삭제 가능

## 📋 시간표 구조

- **교시**: 1~9교시 (9:00 ~ 18:00)
- **요일**: 월요일 ~ 금요일
- **학점 계산**: 1학점 = 1시간 = 1교시

## 💾 저장 방식

모든 강의 정보는 브라우저의 LocalStorage에 자동으로 저장되므로, 페이지를 새로고침해도 데이터가 유지됩니다.

## 🎯 사용 예시

**데이터베이스 (3학점)**
- 월요일: 1~2교시 (2시간)
- 화요일: 3교시 (1시간)
- 총 3시간 = 3학점
